# FPS Aim Trainer - COMS3006A (CGV)

### How to run code
First install dependencies:
```bash
$ npm install
```
This project uses a simple express.js server to run three.js. To start the server run:
```bash
$ npm start
```
Open http://127.0.0.1:3000 to view the project.
